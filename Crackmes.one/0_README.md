Writeups de CTFs sobre Ingeniería Inversa

Repositorio de resoluciones de retos de ingeniería inversa orientados al análisis de binarios, reversing, debugging y explotación de lógica interna. Incluye técnicas de desensamblado, análisis estático y dinámico, bypass de protecciones y comprensión de código a bajo nivel usando herramientas utilizadas en escenarios reales de reversing y seguridad ofensiva
